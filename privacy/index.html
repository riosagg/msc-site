プライバシーポリシー

最終更新日：2025-12-15

1. 収集する情報

本アプリは、提供に必要な範囲で以下の情報を取り扱う場合があります。

  • アカウント情報（Google / Apple ログイン時の識別子 等）
  • アプリ内でユーザーが入力した情報（コメント、ニックネーム 等）
  • 端末情報・アプリ情報（OS、端末モデル、アプリバージョン 等）
  • 利用状況データ（コンテンツ閲覧の記録、クラッシュ/エラー報告 等）
  • 広告関連情報（広告配信に必要な識別子や計測情報 等）

2. 利用目的

  • アプリ機能の提供（コメント表示・投稿、ランキング、ミュート等）
  • 不具合調査・改善（エラー報告、ログ分析）
  • 不正行為対策、セキュリティ確保
  • 無料ユーザー向けの広告表示、および広告の品質改善

3. 外部サービスの利用

本アプリは、機能提供のために外部サービスを利用する場合があります。

  • 認証：Google OAuth / Apple
  • バックエンド：Supabase（Auth / Database / Functions 等）
  • 広告：Google AdMob（Google Mobile Ads）
  • （任意）メール通知：Resend 等

これら外部サービス事業者が、各社の規約・ポリシーに基づきデータを取り扱う場合があります。

4. 第三者提供

法令に基づく場合等を除き、本人の同意なく個人情報を第三者に提供しません。 ただし、広告配信のために広告事業者へ必要情報が送信される場合があります。

5. 広告（AdMob）

無料ユーザー向けに、Google AdMob による広告を表示する場合があります。 プレミアムユーザーには広告を表示しません（アプリ内の状態により制御します）。

app-ads.txt は当サイトのルート（https://meercat.red/app-ads.txt）に配置します。
